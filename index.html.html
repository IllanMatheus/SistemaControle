Configuração Inicial

Passo 1: Criar planilha no Google Sheets

Acesse sheets.google.com e crie uma planilha com 3 abas:

  • Entrada: Data | Tipo | Genero | Tamanho | Quantidade | Estado
  • Saidas: Data | Tipo | Genero | Tamanho | Quantidade | Beneficiario | Voluntario
  • Estoque: Tipo | Genero | Tamanho | Quantidade | Estado

Passo 2: Configurar Apps Script

Na planilha: Extensões > Apps Script

Cole o código fornecido e implante como App Web

Copie a URL gerada (termina com /exec)

Resumo da configuração

Abra o bloco de notas na área de trabalho (sistema doações)

Tem dois arquivos txt, apps_script e link_planilha

Link_planilha vai ir para o google sheets, copie o nome da planilha que tem no bloco de notas

Coloque no campo abaixo

O arquivo apps_script é o arquivo do código que vai dentro da planilha - extensões - app script

Verifique se o código é o mesmo, caso não seja copie e cole o código do bloco de notas apps_script
